input요소
file 속성값이 파일인 요소를 모두 선택택함
submit 속성값이 submit인 요소를 모두 선택함 

getter
선택된 요소에 접근하여 그 값을 읽어오기 위한 매더이며 아무런 인수를 전달하지 않고 호출
setter
선택된 요소에 접근하여 그 값을 설정하기 위한 메서드이며 대입하고자 하는 값을 인수로 전달하여 호출

html,text width heigth,atter,position,val

매서드 체이닝 
getter 선택된 요소가 여러개 존재하면 getter 매서드는 가장 첫 번째 요소 의 값만을 반환함

setter 선택된 모든 요소에 접근할 수 있는 또 다른 제이쿼리 객체를 반환함

매서드 체이닝 도중에 end메서드를 사용하면 바로 이전에 선택했던 요소의 집합을 다시 선택할수있음

width()매서드 height()매서드
선택한 요소의 너비나 높이를 반환하거나 설정

요소추가 
append매서드
선택한 요소의 마지막에 새로운 요소나 콘텐츠를 추가
prepend메서드
선택한 요소의 처음에 새로운 요소나 콘텐츠를 추가 
appendTo
선택한 요소를 해당 요소의 마지막에 삽입 동작은 append와 같지만 소스와 타겟의 위치가 서로 반대
prependTo
선택한 요소를 해당 요소의 마지막에 삽입 동작은 prepend와 같지만 소스와 타겟의 위치가 서로 반대

befoer매서드 
선택한 요소의 바로 앞에 새로운 요소나 콘텐츠를 추가
after메서드
선택한 요소의 바로 뒤에 새로운 요소나 콘텐츠를 추가

insertBefore
선택한 요소를 해당 요소의 앞에 삽입 동작은 같지만 소스와 타겟의 위치가 서로 반대
insertAfter
선택한 요소를 해당 요소의 뒤에 삽입 동작은 같지만 소스와 타겟의 위치가 서로 반대
wrap
선택한 요소를 포함하는 새로운 요소를 추가 
wrapAll
선택한 모든 요소를 포함하는  새로운 요소를 추가 
요소복사
clone
선택한 요소를 복사하여 새로운 요소를 생성append,appendTo와 같이 사용해야함
replaceAll
선택한 요소를 지정된 요소로 대체
replaceWith
선택한 요소를 지정된 요소로 대체
요소삭제
remove
선택한 요소를 dom트리에서 삭제
